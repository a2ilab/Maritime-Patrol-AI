1차: 학습 
1-1: 선박(순찰,경비정)의 경로를 입력받아.
1-2: 주변 정보(조류, 바람, 파도, 선박밀집, 사고다발 등)을 입력받아.
1-3: 선박(순찰,경비정)의 경로와 주변 정보(조류, 바람, 파도, 선박밀집, 사고다발 등)과 연관성을 도출해 내.
1-4: 선박(순찰,경비정)의 경로와 주변 정보(조류, 바람, 파도, 선박밀집, 사고다발 등)과 연관성이 각각의 그래프? 학습모델? 이 나와. 
1-5: 학습은 1회만 이루어지고, 이후에는 학습된 모델만 사용해. 필요할 때만, 새로 학습하도록 할께. 

2차: 사용자 요구사항 입력
2-1: 사용자는 본인의 요구사항을 입력할 수 있어. 
2-2: 주변 정보(조류, 바람, 파도, 선박밀집, 사고다발 등)에 대한 가중치를 설정할 수 있어.
2-3: 그리고 시간과 위치를 입력해서 사용자가 선택한 위치를 추천 지점에 추가할 수 있어.(이건 결정된 순찰지점이라고 생각해야 해)

3차: 추론
3-1: 추론은 Polygon 을 통해 영역을 지정할꺼야. Polygon은 순찰/경비의 책임구역으로 불특정 다각형으로 이루어져 있어. 현재 데이터는 shp 파일로 있어서 내가  geojson변환해서 줄꺼야. 
3-2: 현재의 주변 정보(조류, 바람, 파도, 선박밀집, 사고다발 등)를 입력받아
3-3: 주변 정보(조류, 바람, 파도, 선박밀집, 사고다발 등)를 학습모델에 따라서 추론해서 GRID 형태(격자의 크기는 100미터 기준)의 영향도 맵을 만들어.
3-4: 각각의 주변정보에 대한 영향도 맵을 사용자가 입력한 가중치에 따라서 취합해.
3-5: 취합된 영향도 맵을 통해 추천위치를 선별해
3-6: 데이터는 1시간 단위로 입력될 수 있어. 그래서 방금 추론한 시간 + 1 hour 에 대한 주변정보에 대해서 3-1 부터 다시 추론해.
3-7: 사용자가 설정한 시간의 마지막까지 왔다면, 추론을 종료해.

4. 경로 산출
4-1: 경로 산출은 3.에서 추론한 순찰/경비 추천 지점들을 시간순으로 순회하는 알고리즘이야.
4-2: 3.에서 추론한 순찰/경비 추천 지점에 2-3.에서 사용자가 입력한 위치를 추가해.
4-3: 시간순으로 추천/지정 지점을 순회하는 경로를 계산하는데, Q-Learning 알고리즘을 적용해서 사용해. Q-Learning 알고리즘을 적용할 때는 해당 추천 지점이 도출된 시간의 취합한 영향도맵을 사용하도록 해야 해.  그리고 사용자 지정 지점도 시간에 가장 가까운 취합된 영향도 맵을 사용하도록 해. 만약 중간의 시간이 설정되면, 가장 늦은(이후의)시간대의 취합된 영향도 맵을 사용해야 해.

5. API 결과 전달
5-1: 지금 JSON의 구조를 최대한 유지해. 근데 없는 요소는 추가해야 해. 변경이 꼭 필요한 요소는 알려주고.
