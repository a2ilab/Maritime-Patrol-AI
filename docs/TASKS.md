## 진행 순서 제안

1. **경로 옵션 제거 + 시간 입력** (API inference 고정 파라미터, UI 전략 제거·시간 추가) — 완료 방향.
2. **관계성 레이어** 주석/포맷 정리 (코드 상 포맷 명시).
3. API 스키마·문서 정리 (deprecated 표시 등).
4. **(학습)** 데이터별 그래프 학습 → 그래프별 가중치 학습 → 학습 맵 저장/로드.
5. **(추론)** 그래프 로드 → 시간대 데이터 → 그래프 치환 → grid → 가중치 적용 → 순찰 필요성·설명 누적 → 시간대별 순찰 지역·경로.
6. (추후) GIS 제약 반영.
